# 개인정보처리방침

**알림 전달** (이하 "앱")은 사용자의 개인정보를 중요하게 생각하며, 관련 법령에 따라 아래와 같이 개인정보처리방침을 안내합니다.

**시행일:** 2026년 4월 15일

---

## 1. 수집하는 개인정보 항목

본 앱은 원칙적으로 서버로 개인정보를 수집·전송하지 않습니다. 모든 데이터는 사용자의 기기 내에서만 처리되고 저장됩니다.

앱이 기기 내에서 처리하는 정보는 다음과 같습니다:

| 항목 | 목적 | 저장 위치 |
|------|------|-----------|
| 알림 데이터 (앱 이름, 제목, 내용, 시간, 이미지) | 알림 기록 저장 및 전달 규칙 매칭 | 기기 내부 DB |
| 전달 규칙 (대상 앱, 키워드, 수신 번호, URL 등) | 사용자가 설정한 자동 전달 조건 관리 | 기기 내부 DB |
| 전달 이력 (발송 성공/실패 기록) | 전달 결과 확인 | 기기 내부 DB |

단, **클라우드 백업** 기능을 사용할 경우 사용자의 명시적 요청에 의해 전달 규칙 설정이 서버에 임시 업로드됩니다. 해당 데이터는 72시간 후 자동 삭제됩니다.

## 2. 개인정보의 보유 및 이용 기간

- 알림 데이터 및 전달 로그는 **1개월** 보관 후 자동 삭제됩니다.
- 전달 규칙은 사용자가 직접 삭제하기 전까지 유지됩니다.
- 클라우드 백업 데이터는 업로드 후 **72시간** 이내에 서버에서 자동 삭제됩니다.
- 앱을 삭제하면 기기에 저장된 모든 데이터가 함께 삭제됩니다.

## 3. 개인정보의 제3자 제공

본 앱은 사용자의 개인정보를 외부 서버로 임의 전송하거나 제3자에게 제공하지 않습니다.

단, 사용자가 직접 설정한 다음 기능은 사용자의 명시적 의사에 따라 기기 외부로 데이터가 전송될 수 있습니다:

- **SMS 자동 발송**: 사용자가 지정한 전화번호로 문자 메시지를 발송합니다.
- **URL 호출**: 사용자가 지정한 외부 URL로 HTTP 요청을 전송합니다.
- **Telegram 전송**: 사용자가 설정한 Telegram 봇을 통해 메시지를 전송합니다.
- **Discord 전송**: 사용자가 설정한 Discord 웹훅 URL로 메시지를 전송합니다.
- **Slack 전송**: 사용자가 설정한 Slack 웹훅 URL로 메시지를 전송합니다.
- **클라우드 백업**: 사용자의 요청 시 전달 규칙 설정을 서버에 임시 저장합니다 (72시간 후 자동 삭제).

이러한 기능은 사용자가 직접 설정한 경우에만 작동하며, 앱이 자체적으로 외부에 데이터를 전송하지 않습니다.

## 4. 광고

본 앱은 Google AdMob을 통해 광고를 표시합니다. AdMob은 광고 제공을 위해 광고 식별자(Advertising ID) 등의 정보를 수집할 수 있으며, 이는 [Google 개인정보처리방침](https://policies.google.com/privacy)에 따릅니다.

## 5. 앱 권한

본 앱은 다음 권한을 사용하며, 모든 권한은 앱의 핵심 기능 수행을 위해서만 사용됩니다:

| 권한 | 용도 |
|------|------|
| 알림 접근 (Notification Listener) | 다른 앱의 알림을 읽어 전달 규칙과 매칭 |
| SMS 발송 (SEND_SMS) | 조건 충족 시 자동 문자 발송 |
| 인터넷 (INTERNET) | URL 호출, Telegram/Discord/Slack 전송, 클라우드 백업, 광고 표시 |
| 부팅 시 자동 시작 (BOOT_COMPLETED) | 기기 재부팅 후 서비스 자동 재시작 |
| 포그라운드 서비스 | 백그라운드에서 안정적인 알림 감시 |

## 6. 이용자의 권리

- 사용자는 언제든지 앱 내에서 저장된 알림 기록, 전달 규칙, 전달 로그를 확인·수정·삭제할 수 있습니다.
- 앱 삭제 시 기기에 저장된 모든 데이터가 완전히 삭제됩니다.
- 기기 설정에서 알림 접근 권한을 해제하면 알림 수신이 즉시 중단됩니다.

## 7. 개인정보의 안전성 확보 조치

- 수신 번호, Bot Token, Webhook URL 등 민감한 정보는 기기 내 암호화(AES-256-GCM)되어 저장됩니다.
- 클라우드 백업 데이터는 HTTPS를 통해 전송되며, 72시간 후 서버에서 자동 삭제됩니다.
- 외부 서비스(Telegram, Discord, Slack) 전송은 HTTPS만 허용합니다.
